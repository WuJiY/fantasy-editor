## Installation

### npm

```bash | pure
npm install fantasy-editor --save
```

## Examples

```bash | pure
# Clone the repository
$ git clone https://github.com/lingjieee/fantasy-editor.git

# Change directory
$ cd fantasy-editor

# Install dependencies
$ yarn install

# Run examples
$ yarn start
```
