</div>

## 安装

### npm

```bash | pure
npm install fantasy-editor --save
```

## 示例

```bash | pure
# 克隆仓库
$ git clone https://github.com/lingjieee/fantasy-editor.git

# 切换目录
$ cd fantasy-editor

# 安装依赖
$ yarn install

# 运行示例
$ yarn start
```
